
### Usage

Deploy lab with command
```
./deploy.sh <lab>
```
Destroy lab with command
```
./destroy.sh <lab>
```

Access server console with command
```
./server.sh <lab> <server>
```
for example
```
./server.sh lab1 snmp
```

Access router CLI with command
```
./router.sh <lab> <router_number>
```
